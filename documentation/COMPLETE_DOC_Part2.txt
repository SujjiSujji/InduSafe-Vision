1. YOLOv8 Object Detection:

Architecture:
- Backbone: CSPDarknet with cross-stage partial connections
- Neck: PAN-FPN (Path Aggregation Network + Feature Pyramid Network)
- Head: Decoupled head with anchor-free detection
- Loss Functions: CIoU loss, DFL (Distribution Focal Loss)

Inference Process:
```python
# Load model
model = YOLO('yolov8n.pt')

# Preprocess frame
frame_rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)

# Run inference
results = model(frame, verbose=False)

# Extract detections
for result in results:
    boxes = result.boxes
    for box in boxes:
        x1, y1, x2, y2 = box.xyxy[0]
        confidence = box.conf[0]
        class_id = int(box.cls[0])
```

Performance Optimization:
- Nano model variant (YOLOv8n) for speed
- Batch size = 1 for real-time
- FP16 inference on GPU (optional)
- Multi-threaded frame processing

2. QR Code Detection (OpenCV):

Algorithm: cv2.QRCodeDetector.detectAndDecodeMulti()

Process:
```python
detector = cv2.QRCodeDetector()
ret, decoded_info, points, _ = detector.detectAndDecodeMulti(frame)

if ret and decoded_info:
    for i, data in enumerate(decoded_info):
        if data:
            bbox = points[i]
            # Extract bounding box coordinates
            x = min([int(p[0]) for p in bbox])
            y = min([int(p[1]) for p in bbox])
            w = max([int(p[0]) for p in bbox]) - x
            h = max([int(p[1]) for p in bbox]) - y
```

Error Correction: Reed-Solomon coding (Level M, 15% recovery capability)

3. Spatial Matching Algorithm:

Purpose: Associate detected persons with QR codes

Algorithm:
```python
def match_worker(person_bbox, qr_codes):
    v_center_x = (person_bbox[0] + person_bbox[2]) / 2
    v_center_y = (person_bbox[1] + person_bbox[3]) / 2
    
    best_match = None
    min_distance = float('inf')
    
    for qr in qr_codes:
        qr_center_x = qr['x'] + qr['width'] / 2
        qr_center_y = qr['y'] + qr['height'] / 2
        
        # Euclidean distance
        distance = sqrt((v_center_x - qr_center_x)**2 + 
                       (v_center_y - qr_center_y)**2)
        
        # Prefer QR inside person bbox
        if is_inside(person_bbox, qr):
            distance *= 0.5
        
        if distance < min_distance:
            min_distance = distance
            best_match = qr
    
    return best_match if min_distance < 300 else None
```

Complexity: O(n*m) where n=persons, m=QR codes

4. Threshold Logic (False Positive Reduction):

Consecutive Frame Counting:
```python
violation_counters[worker_key] += 1

if violation_counters[worker_key] >= THRESHOLD:
    # Confirmed violation
    trigger_alert()
    violation_counters[worker_key] = 0
else:
    # Wait for more frames
    continue
```

Optimal Threshold: 3 frames (~100ms at 30 FPS)
False Positive Reduction: ~70%
